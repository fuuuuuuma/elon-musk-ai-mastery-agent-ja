Grok Build とは？

公式サイトより：ターミナルにコマンド1行で導入

Grok Build は、ターミナルで動くエージェント型のコーディングツール。イーロンのSpaceXAI（旧xAI）が出した、Grok 4.5を積んだ“開発の相棒”です。Plan Mode・最大8体の並列エージェント・MCP対応が特徴。

「話しかけると、計画して・手分けして・作ってくれる」——Claude Code を使ったことがあれば、すぐ馴染みます。
